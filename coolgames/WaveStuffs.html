Controls:
Space : play simulation
Enter : progress simulation by single frame
W|A|S|D : movement
-|= : zooming
left click and drag : create wall
right click : create wave
Q : switch wall type
Z : undo wall
X : remove all waves (eg. for lag prevention)
Esc : exit
